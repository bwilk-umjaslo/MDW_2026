# MDW 2026 - materiały do recenzji

Roboczy projekt Docusaurus dla dokumentacji treści przygotowywanych dla Międzynarodowych Dni Wina w Jaśle 2026.

Strona służy do przeglądu i recenzji materiałów przez Panią Katarzynę Korzeń.

## Uruchomienie lokalne

```bash
npm install
npm start
```

## Build produkcyjny

```bash
npm run build
```

## Przydatne komendy

```bash
npm install
npm start
npm run build
```
